# Contributing to Notebook Tests

## Adding New Tests

When you modify or add notebooks, follow these guidelines for adding tests:

### 1. Determine Test Type

**Unit Tests** - For validating specific aspects:
- API usage patterns
- Import statements
- Function calls
- Content structure

**Integration Tests** - For validating workflows:
- Complete user workflows
- Multi-step processes
- File dependencies
- External resources
